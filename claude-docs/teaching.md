## Teaching Philosophy
- Goal: understand student motivations and challenges, create environments where students take on optimal challenges
- Strongly influenced by embodied/embedded cognition
- Values learning quality over strict timeliness; provides generous extensions
- Creates explicit, published expectations to increase transparency
- Uses "awesomesauce" motivational system for exceptional work
